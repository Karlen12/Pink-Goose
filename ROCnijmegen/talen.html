Java is een programmeertaal waarmee je allerlei soorten software kunt maken, van apps op je telefoon tot grote systemen bij bedrijven. Het is alsof je recepten schrijft voor de computer, zodat hij precies weet wat hij moet doen. Java is bijzonder omdat een programma dat je ermee maakt op bijna elke computer kan draaien, of het nu een Windows-pc, Mac, server of telefoon is. Het is ook erg betrouwbaar en wordt al tientallen jaren gebruikt in bedrijven, banken en webservices. Zonder Java zou veel van de software die we dagelijks gebruiken gewoon niet bestaan.
HTML is de basis van een website, het bepaalt wat er op de pagina staat. Je gebruikt het om titels, tekst, afbeeldingen en links te plaatsen. Het is als het skelet of de muren van een huis: het geeft structuur, maar geen stijl. Zonder HTML zou een pagina leeg zijn, want de browser weet dan niet wat hij moet laten zien.
CSS bepaalt hoe een website eruitziet. Het kiest kleuren, lettertypes, groottes en posities van de elementen. Je kunt het vergelijken met verf, meubels en decoratie in een huis: het maakt alles mooi en overzichtelijk. Zonder CSS ziet een pagina er saai en standaard uit, maar de structuur van HTML blijft zichtbaar.
